How to run the Expense Manager Project :--

1.> Download the zip file.

2.> Extract the file and copy MyProject folder.

3.> Paste inside root directory wamp64/www.

4.> Start the WAMP server.

5.> Open PHPMyAdmin (http://localhost/phpmyadmin)

6.> Create a database with name expense_manager. 

7.> Import expense_manager.sql file (given inside the zip package in Database file folder)

8.> Run the script http://localhost/MyProject (frontend)
